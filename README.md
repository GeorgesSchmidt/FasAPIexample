## Description

**NotesFastAPI** is a lightweight web application for creating and displaying notes, built with **FastAPI** and **SQLite**, and powered by a minimal HTML frontend.

This project is great for:
- showcasing backend development skills,
- demonstrating containerization with Docker,
- presenting a clean, simple, and functional app in a portfolio.

---

## Features

- **FastAPI** backend  
- **SQLite** database  
- Simple HTML frontend (Jinja2)  
- Auto-generated interactive API docs (Swagger UI)  
- Full Docker support for easy deployment
